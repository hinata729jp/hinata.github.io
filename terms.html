利用規約

最終更新日：2025年5月31日

トップページに戻る

本利用規約（以下「本規約」）は、ヒナタAI（以下「本サービス」）の利用条件を定めるものです。本サービスをご利用いただく前に、本規約をよくお読みください。本サービスへのお申し込みをもって、本規約に同意したものとみなします。

第1条（サービス内容）

本サービスは、LINE上で利用できるAIカウンセリングサービスです。ユーザーのMBTIタイプと選択した相談モード（寄り添い・分析・正論）に基づき、AIが回答を提供します。本サービスはヒナタのYouTube動画の内容を学習したAIによるものであり、医療行為、心理カウンセリング、法律相談、その他の専門的助言を提供するものではありません。

第2条（利用料金）

  1. 月額料金：1,480円（税込）
  2. 支払い方法：クレジットカード決済（Stripe経由）
  3. 課金サイクル：申込日を起算日とし、毎月同日に自動更新・自動課金されます
  4. 無料トライアル：初回登録から1週間は無料トライアル期間です。無料期間中に解約した場合、料金は発生しません。期間終了後は自動的に通常料金（1,480円）が課金されます

第3条（解約）

  1. ユーザーはいつでも解約手続きを行うことができます
  2. 解約方法：LINE上で「解約」と送信すると案内されるStripeカスタマーポータルから手続きできます
  3. 解約後も、現在の請求期間の終了日まで本サービスを利用できます
  4. 解約手続き完了後、次回以降の課金は発生しません

第4条（返金ポリシー）

本サービスはデジタルコンテンツの性質上、原則として返金には応じかねます。ただし、システム障害等により本サービスが長期間利用できなかった場合は、個別にご相談ください。

第5条（禁止事項）

ユーザーは以下の行為を行ってはなりません。

  1. 本サービスを商業目的で利用する行為
  2. 本サービスの回答を自己の著作物として公開する行為
  3. 本サービスのシステムに対する不正アクセスや妨害行為
  4. 他のユーザーのアカウントを使用する行為
  5. その他、運営者が不適切と判断する行為

第6条（免責事項）

  1. 本サービスのAIによる回答は参考情報であり、その正確性・完全性を保証するものではありません
  2. 本サービスの利用により生じた損害について、運営者は法令上許容される範囲で責任を負いません
  3. 緊急性のある悩み（自傷・他害のおそれ等）については、専門の相談窓口や医療機関にご相談ください
  4. システムメンテナンスや障害により、一時的にサービスが利用できない場合があります

第7条（個人情報の取扱い）

  1. 本サービスで取得する情報：LINEユーザーID、相談内容、MBTIタイプ、決済情報（Stripe経由）
  2. 利用目的：サービス提供、課金管理、サービス改善
  3. 第三者提供：法令に基づく場合を除き、ユーザーの同意なく第三者に提供しません
  4. 決済情報はStripe社が管理し、運営者がクレジットカード番号を直接保有することはありません

第8条（規約の変更）

運営者は本規約を変更する場合があります。重要な変更がある場合は、LINE公式アカウントを通じて事前に通知します。変更後も本サービスの利用を継続した場合、変更後の規約に同意したものとみなします。

第9条（準拠法・管轄）

本規約は日本法に準拠し、本サービスに関する紛争は、運営者の所在地を管轄する裁判所を第一審の専属的合意管轄裁判所とします。
